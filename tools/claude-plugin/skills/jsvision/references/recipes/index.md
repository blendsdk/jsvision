# Recipes

Complete, runnable app patterns — one per archetype. Each page embeds a literal copy of a real,
smoke-tested module under `packages/examples/recipes/`, so the code you read is the code that runs.
Pick the closest archetype, copy its module, and adapt it.

| Archetype                    | Page                                   | Module                      |
| ---------------------------- | -------------------------------------- | --------------------------- |
| Data-driven & master-detail  | [data-driven.md](data-driven.md)       | `recipes/data-grid.ts`      |
| Forms, dialogs & validation  | [forms-dialogs.md](forms-dialogs.md)   | `recipes/form-dialog.ts`    |
| File & text tools            | [file-text.md](file-text.md)           | `recipes/file-tools.ts`     |
| Live / dashboard (+ browser) | [live-dashboard.md](live-dashboard.md) | `recipes/live-dashboard.ts` |

To author a custom widget instead, see [../widget-authoring.md](../widget-authoring.md).
