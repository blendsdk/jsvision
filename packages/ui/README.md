# @jsvision/ui

The **widget framework** of [jsvision](https://github.com/blendsdk/jsvision) — a
retained widget tree with fine-grained signal reactivity, built on the
[`@jsvision/core`](../core) engine. Windows, menus, dialogs, lists, tables, tabs,
trees, a text editor, and ~40 controls in the classic Turbo Vision style.

> **Pre-release (0.x), under heavy development.** The public API may still change
> before 1.0 — pin an exact version.

```bash
npm install @jsvision/core @jsvision/ui
```

```ts
import { createApplication, Window, Button } from '@jsvision/ui';
```

Everything is re-exported from the single entry point (`@jsvision/ui`).

See the [documentation site](https://blendsdk.github.io/jsvision/) for the
[guide](https://blendsdk.github.io/jsvision/guide/), the live
[component gallery](https://blendsdk.github.io/jsvision/components/), and the
[API reference](https://blendsdk.github.io/jsvision/api/).
